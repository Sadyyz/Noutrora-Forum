# NOUTRORA FORUM

Projeto simples de interface para estudo de layout.

## O que é
- Tela inicial
- Tela de login
- Tela de registro

## Como executar
1. Instale as dependências do Flask:
   ```bash
   pip install flask
   ```
2. Inicie o servidor:
   ```bash
   python app.py
   ```
3. Abra no navegador:
   ```text
   http://127.0.0.1:5000/
   ```

## Observação
Este projeto está apenas no **layout UI**. Não há autenticação, API ou banco de dados conectados.
